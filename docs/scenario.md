# Сценарий

## 1 кадр

Огромная башня из [дженги](https://ru.wikipedia.org/wiki/Дженга) возвышается посередине кадра. Башня состоит из 11 этажей. Каждый этаж состоит из трёх блоков, положенных вплотную параллельно друг другу. Блоки каждого следующего этажа расположены перпендикулярно блокам предыдущего этажа. На 5 этаже расположен только один блок посередине.

Аутскаут схватился за один блок с пятого этажа и тянет на себя. Рост Аутскаута равен высоте одного блока. Он упирается ногами на четвертый этаж.

Аутскаут в это время говорит: «Конец несбалансированной системы...»

## 2 кадр

Башня разбита вдребезги. Ее блоки свалены в кучу.

Аутскаут стоит рядом с кучей. Он продолжает мысль: «Наступает быстро и громко!». При этом вытирает выступивший пот платком. Рядом с ним витают мысли: «Фух, блин. Едва справился».
