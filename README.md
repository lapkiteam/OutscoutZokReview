# OutscoutZokReview

![OutscoutUnbalancedSystem](https://lapkiteam.github.io/OutscoutZokReview/OutscoutUnbalancedSystem.png)

*Сценарий комикса — [тут](./docs/scenario.md).*

Вышел видеообзор «Сокрытое в Зимней Олимпиаде Квестов» от Аутскаута! Видео доступно [в VK Видео](https://vkvideo.ru/video-235382942_456239021) и [на YouTube](https://www.youtube.com/watch?v=pG1NSObelU8).

Из обзора усвоил следующее:

* Зря не прошел «[Башню Парабеда](https://db.crem.xyz/game/3435/)». Кто ж знал, что башня настолько высокая, а смысол настолько глубокий 🔥
* Анализ «[Психотленного](https://db.crem.xyz/game/3447/)» оказался лучше самой игры. Надо совершенствоваться для соответствия 😖
* «[Амба](https://db.crem.xyz/game/3436/)» — в первую очередь хобана, но вот ее подоплека…

Юмор на серьезных щах и глубокий экономический анализ под веселую музычку прилагается. Смотреть обязательно.
